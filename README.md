## 特点

> 部分为专业版能力

- AI 友好的工程底座，内置多个 Skills
- 一流的技术栈：Vue 3.6 / Vite 8 / Pinia / UnoCSS / VueUse / TypeScript / ESLint / Stylelint / ...
- 自由选择喜爱的 UI 库，默认 Vant
- 核心布局组件，提供顶部导航栏、顶部标签栏、返回顶部、记录滚动位置等特性
- 基于文件系统的路由
- 精细可控的页面保活策略
- 全方位权限验证
- 国际化、RTL支持
