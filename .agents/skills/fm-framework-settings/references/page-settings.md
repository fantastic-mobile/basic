# 页面设置 (page)

## progress

- **类型**: `boolean`
- **默认值**: `true`
- **说明**: 是否显示页面载入进度条

## backTop

- **类型**: `boolean`
- **默认值**: `true`
- **说明**: 是否显示返回顶部按钮

## 完整示例

```typescript
page: {
  progress: true,
  backTop: true,
}
```

## 常见配置

**禁用进度条和返回顶部:**
```typescript
page: {
  progress: false,
  backTop: false,
}
```
